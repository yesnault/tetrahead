Avec cette barre d'outils, vous pourrez lancer (ou arrêter)
l'exécution du synthtétiseur. Elle comporte trois boutons :
\begin{itemize}
\item Un bouton \emph{Lecture} : En cliquant sur ce bouton, vous
démarrerez l'exécution du synthétiseur.
\item Un bouton \emph{Arrêt} : Un clic sur ce bouton stoppe
l'exécution du synthétiseur.

\end{itemize}
