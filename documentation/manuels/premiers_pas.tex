\section{Premiers pas}

Cette partie du manuel explique comment créer un premier montage avec
quelques modules.

\subsection*{\'Etape 1}

Démarrer \tth pour afficher l'écran principal.

\subsection*{\'Etape 2 : Création de modules}

Dans la barre d'outils en bas de l'écran, se trouvent les boutons de création 
de modules. Cliquer (mais sans relacher le bouton de la souris) sur un bouton 
\emph{VCO} de la barre d'outils inférieure :
\begin{center}
	\includegraphics{tutorial/vco.png}
\end{center}

Puis déplacer la souris dans l'espace de travail. Un module est apparu à 
l'écran. Il peut être déplacé n'importe où dans l'espace de travail.

Recommencer l'opération avec un module \emph{OUTDirect} :
\begin{center}
	\includegraphics{tutorial/out.png}
\end{center}

Pour savoir quels sont les rôles de ces modules, se reporter à la 
section~\ref{secDescModules}, page~\pageref{secDescModules}.
L'écran doit ressembler à ceci :
\begin{center}
	\includegraphics[width=\textwidth]{tutorial/etape2.png}
\end{center}

\subsection*{\'Etape 3 : Connexion de modules}

Pour générer du son, il faut connecter ces deux modules ensemble en réalisant 
un glisser-déplacer avec la souris depuis un port de sortie vers un port 
d'entrée.
\begin{center}
	\includegraphics[width=\textwidth]{tutorial/etape3.png}
\end{center}

\info{Pendant le glisser-déplacer, une bordure verte apparaît autour du port 
d'entrée pour indiquer que la connexion est autorisée. Si le port d'entrée 
était déjà connecté, la connexion aurait été refusée et la bordure serait rouge.
}

\subsection*{Étape 4 : Modification d'un paramètre discret}

Le VCO compte deux paramètres : un discret, à droite, et un continu à gauche. 
Cliquer 4 fois avec le bouton droit de la souris sur le bouton \emph{Shapes}
du VCO. Cela fait tourner le bouton du paramètre d'un cran vers la droite à 
chaque clic droit (réciproquement avec un clic gauche). La courbe représentée 
doit normalement être une sinusoïde :
\begin{center}
	\includegraphics[scale=0.7]{tutorial/etape4.png}
\end{center}

\subsection*{Étape 5 : Modification d'un paramètre continu}

Cliquer (sans relacher) sur le bouton \emph{Pitch} du VCO. Cela modifie la 
valeur du paramètre, valeur que l'on peut voir dans l'étiquette au-dessus du 
bouton.

Tourner le bouton jusqu'à obtenir une fréquence de $440Hz$. Avec la souris, il 
est difficile d'obtenir un résultat aussi précis. Pour y remédier, il faut 
utiliser le clavier. Sélectionner tout le contenu de l'étiquette :

Saisir \emph{440} au clavier, puis taper \emph{Entrée} :
\begin{center}
	\includegraphics[scale=0.7]{tutorial/etape5-2.png}
\end{center}

\subsection*{Étape 6 : Synthèse du son}

Ce montage est un montage minimum pour produire du son. Cliquer sur le bouton 
de lecture (triangle vert dans la barre d'outils en haut de la fenêtre) pour 
commencer la synthèse du son. Le bouton rouge est le bouton d'arrêt.

Avec ce montage, le son produit est un La comme on peut l'obtenir avec un 
diapason ou en décrochant son téléphone.

\subsection*{Étape 7 : Suppression d'un fil}

Arrêter la synthèse du son. Cliquer avec le bouton droit de la souris sur le fil
connectant les deux modules pour le supprimer.

\attention{Il n'est possible ni de supprimer un fil, ni d'ajouter ou supprimer des 
modules pendant la synthèse de son.}

\subsection*{Étape 8 : Suppression d'un module}

Pour supprimer un module, il suffit de le jeter dans la corbeille à papiers présente sur l'espace de travail.

\info{La corbeille, comme les modules, est déplaçable dans l'espace de travail. 
On peut donc la placer où l'on veut.}
