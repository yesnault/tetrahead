\section{Description}

Ce document rassemble les compte-rendus de l'ensemble des tests réalisés sur la partie \emph{Core.Modules} de l'application.
\smallskip
Un compte-rendu type comporte :
\begin{itemize}
 \item (en titre de section) la personne et la date de la série de tests
 \item la liste des tests effectués
 \item le cas échéant, les erreurs corrigées
\end{itemize}
